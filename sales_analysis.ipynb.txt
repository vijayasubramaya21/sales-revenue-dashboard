    "## 2. Clean the Data\n",
    "\n",
    "Issues found and how each is handled:\n",
    "\n",
    "| Issue | Fix |\n",
    "|---|---|\n",
    "| Duplicate order rows | Drop exact duplicates |\n",
    "| Inconsistent region text (` north`, `NORTH`, `North `) | Strip whitespace + title-case |\n",
    "| Missing `region` | Drop rows \u2014 can't attribute revenue without geography |\n",
    "| Missing `discount_pct` | Fill with 0 (no discount recorded = none applied) |\n",
    "| Negative `quantity` (data entry errors) | Drop \u2014 not valid sales |\n",
    "| `order_date` stored as text | Convert to datetime |"
   ]
  },
  {
   "cell_type": "code",
   "execution_count": null,
   "metadata": {},
   "outputs": [],
   "source": [
    "clean = df.copy()\n",
    "\n",
    "# 1. Drop exact duplicate rows\n",
    "before = len(clean)\n",
    "clean = clean.drop_duplicates()\n",
    "print(f\"Dropped {before - len(clean)} duplicate rows\")\n",
    "\n",
    "# 2. Standardize region text\n",
    "clean[\"region\"] = clean[\"region\"].str.strip().str.title()\n",
    "\n",
    "# 3. Drop rows with missing region (can't analyze geography without it)\n",
    "before = len(clean)\n",
    "clean = clean.dropna(subset=[\"region\"])\n",
    "print(f\"Dropped {before - len(clean)} rows with missing region\")\n",
    "\n",
    "# 4. Fill missing discount_pct with 0\n",
    "clean[\"discount_pct\"] = clean[\"discount_pct\"].fillna(0)\n",
    "\n",
    "# 5. Remove invalid quantity entries\n",
    "before = len(clean)\n",
    "clean = clean[clean[\"quantity\"] > 0]\n",
    "print(f\"Dropped {before - len(clean)} rows with invalid quantity\")\n",
    "\n",
    "# 6. Convert date column\n",
    "clean[\"order_date\"] = pd.to_datetime(clean[\"order_date\"])\n",
    "clean[\"month\"] = clean[\"order_date\"].dt.to_period(\"M\")\n",
    "clean[\"quarter\"] = clean[\"order_date\"].dt.to_period(\"Q\")\n",
    "\n",
    "print(f\"\\nFinal clean dataset: {len(clean):,} rows ({len(df) - len(clean)} removed, {(len(df)-len(clean))/len(df):.1%} of raw data)\")\n",
    "clean.to_csv(\"clean_sales_data.csv\", index=False)\n",
    "clean.head()"
   ]

    "print(cat_quarter)\n",

    "ax.set_ylabel(\"Net Revenue ($)\")\n",
    "ax.set_title(\"Quarterly Revenue by Category \u2014 2025\", fontsize=13, fontweight=\"bold\")\n",
    "ax.yaxis.set_major_formatter(mticker.FuncFormatter(lambda x, _: f\"${x:,.0f}\"))\n",
    "ax.legend(loc=\"upper left\", fontsize=9)\n",
    "plt.tight_layout()\n",
    "plt.savefig(\"chart_category_quarterly_trend.png\", dpi=150)\n",
    "plt.show()"

   "cell_type": "code",

   "source": [
    "# Quantify the decline: Q1+Q2 vs Q3+Q4 for each category\n",
    "clean[\"half\"] = clean[\"order_date\"].dt.month.apply(lambda m: \"H1 (Jan-Jun)\" if m <= 6 else \"H2 (Jul-Dec)\")\n",
    "half_compare = clean.groupby([\"category\", \"half\"])[\"net_revenue\"].sum().unstack(\"half\")\n",
    "half_compare[\"pct_change\"] = (half_compare[\"H2 (Jul-Dec)\"] - half_compare[\"H1 (Jan-Jun)\"]) / half_compare[\"H1 (Jan-Jun)\"]\n",
    "half_compare = half_compare.sort_values(\"pct_change\")\n",
    "\n",
    "print(half_compare)\n",
    "print()\n",
    "worst = half_compare.index[0]\n",
    "decline_pct = half_compare.loc[worst, \"pct_change\"]\n",
    "print(f\"INSIGHT: '{worst}' revenue {'declined' if decline_pct < 0 else 'grew'} \"\n",
    "      f\"{abs(decline_pct):.1%} from H1 to H2 2025 \u2014 the steepest move of any category.\")"

    "## 7. Key Findings\n",
    "\n",
    "- **Office Supplies is the only category to shrink across the year**, down roughly 5% comparing H1 to H2 2025, while every other category grew. The effect is concentrated in Q3-Q4 \u2014 quarterly revenue for the category peaks in Q2 and slides afterward. This is the kind of early warning a category manager would want flagged before it compounds into next year.\n",
    "- **Electronics shows the strongest acceleration**, up 13% from H1 to H2, with a visible Q4 lift consistent with holiday demand \u2014 useful for planning inventory and staffing ahead of next year's Q4.\n",
    "- **Online is the dominant sales channel** by both revenue and order volume, with a higher average order value than Partner Reseller \u2014 useful context if marketing spend is being allocated across channels.\n",
    "- **Profit margin is consistent at ~42% across every region and channel** in this dataset, which means revenue differences between regions reflect volume, not pricing or discounting differences \u2014 a useful sanity check before assuming any region is \"underperforming\" on profitability."
